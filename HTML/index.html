Etiquetas Semanticas Estructurales 游녡

header - Cabecera de un siteo web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento.
footer - Pie de pagina de un sitio web o de una seccion
nav - Representa una navegacion
article - Representa una seccion autocontenido (que por si solo se explica)
aside - Representa contenido complementario o secundario
section - Representa una seccion de contenido generico
address - Representa una informacion de contacto

Etiquetas de bloques vs etiquetas de linea 游녡

La etiqueta de bloque mas representativa es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas.
La etiqueta de linea mas representativa es la span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido. Hola soy otra span

Estilos CSS en HTML 游녡

1. Con el atributo style=""

2. Con la etiqueta style: h2{ color: red; } (en el head).

3. Creando un nuevo archivo de extension .css, enlazando el archivo con la etiqueta link: link rel="stylesheet" href="ruta.css" /link>
